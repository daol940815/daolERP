- 조사: 은행거래에 "하나카드/카드" 결제 출금 **0건**. 카드 사용내역 3건(672,700원)은 전부 미결제 상태로 미지급금이 정확히 남아있음.

결산조정·대체전표 등 직접 분개 입력.

**구현**
- `POST /api/journal/manual`: 균형검증(차=대, >0) + `post_journal`(source_type=manual, 멱등 채번) 호출. `source_id`는 서버 생성 UUID.
- `DELETE /api/journal/manual?sourceId=`: manual 출처만 삭제 허용(안전장치)
- 분개장 화면: "+ 수동 분개" 버튼 → 입력 모달(일자·적요·다중 라인·차대 균형 표시), 수동 전표에 삭제 버튼

**라이브 검증 (운영 DB)**
- 생성: `(차)보통예금 1,000 / (대)잡이익 1,000` → `JV-20260629-0001` 정상 생성 ✅
- 불균형 거부: `(차)1,000 / (대)500` → `차변(1,000) ≠ 대변(500)` 거부 ✅
