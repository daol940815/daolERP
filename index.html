☁️ 앱-웹 연동 설정
PC와 휴대폰에서 같은 데이터를 실시간으로 공유하려면 아래 Supabase 정보를 입력하세요. 모든 기기에 동일한 URL·Key·공유코드를 입력하면 서로 연동됩니다.
📖 처음이신가요? Supabase 설정 방법 (약 3분, 무료)
  1. supabase.com 가입 후 New project 생성
  2. 왼쪽 메뉴 SQL Editor → 아래 코드 붙여넣고 Run
    3. create table if not exists inventory (
  room text not null,
  date text not null,
  slot text not null default 'main',
  data jsonb not null default '[]',
  updated_at timestamptz not null default now(),
  updated_by text,
  primary key (room, date, slot)
);
create table if not exists inventory_config (
  room text primary key,
  products jsonb not null default '[]',
  updated_at timestamptz not null default now(),
  updated_by text
);
create table if not exists inventory_log (
  lid text primary key,
  room text not null,
  date text not null,
  ts timestamptz not null default now(),
  product text,
  detail jsonb,
  updated_by text
);
alter table inventory enable row level security;
alter table inventory_config enable row level security;
alter table inventory_log enable row level security;
create policy "all inv" on inventory for all using (true) with check (true);
create policy "all cfg" on inventory_config for all using (true) with check (true);
create policy "all log" on inventory_log for all using (true) with check (true);
alter publication supabase_realtime add table inventory;
alter publication supabase_realtime add table inventory_config;
alter publication supabase_realtime add table inventory_log;
  3. 왼쪽 Project Settings → API → Project URL과 anon public key 복사
  4. 아래에 입력. Project URL은 https://xxx.supabase.co 형태 (뒤에 /rest/v1 등은 빼세요 — 자동 정리됨)
  5. 공유코드는 팀에서 정한 아무 단어(예: daol-창고)를 모든 기기에 동일하게 입력
같은 공유코드를 쓰는 기기끼리 데이터가 연동됩니다.
