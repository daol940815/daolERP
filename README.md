# daolERP — (주)스피어스 자회사 종합관리 웹

기존 엑셀로 관리하던 **자회사 종합관리 문서**를 웹 시스템으로 옮긴 프로젝트입니다.
세금계산서 · 카드 사용내역 · 통장 입출금 등 **로우데이터 입력/조회**와 **엑셀 가져오기**를 제공합니다.

> 데이터는 `prisma/erp.db` 파일 하나에 저장됩니다. 이 파일만 복사하면 백업, 통째로 옮기면 이전입니다. (엑셀 파일 하나 관리하던 것과 동일한 느낌)

---

## 🖥️ 처음 실행하기 (비개발자용 단계별 가이드)

### 1단계. Node.js 설치 (최초 1회)

1. 인터넷에서 **[https://nodejs.org](https://nodejs.org)** 접속
2. 가운데 **LTS** 버튼(권장 버전)을 눌러 설치 파일 다운로드
3. 다운로드한 파일을 실행해서 **"다음 → 다음 → 설치"** 클릭 (특별한 설정 없이 그대로 진행)
4. 설치 완료

> 설치가 잘 됐는지 확인하려면: Windows는 **명령 프롬프트(cmd)**, Mac은 **터미널**을 열고
> `node -v` 를 입력했을 때 `v20...` 같은 버전이 나오면 성공입니다.

### 2단계. 프로젝트 폴더에서 명령 프롬프트(터미널) 열기

- **Windows**: 프로젝트 폴더(daolERP)에서 주소창에 `cmd` 입력 후 Enter
- **Mac**: 폴더에서 우클릭 → "폴더에서 새로운 터미널 열기"

### 3단계. 최초 1회 준비 (아래 명령을 한 줄씩 입력)

```bash
npm install        # 필요한 프로그램 자동 설치 (1~2분)
npm run db:setup   # 데이터 저장소(데이터베이스) 생성
```

### 4단계. 실행

```bash
npm run dev
```

실행되면 화면에 `http://localhost:3000` 주소가 나옵니다.
**웹 브라우저(크롬 등)를 열고 주소창에 `http://localhost:3000` 을 입력**하면 시스템이 열립니다.

> 종료하려면 명령 프롬프트 창에서 `Ctrl + C` 를 누르세요.
> 다음부터는 **4단계(`npm run dev`)만** 실행하면 됩니다.

## 📥 기존 엑셀 데이터 가져오기

1. 왼쪽 메뉴에서 **⬆️ 엑셀 가져오기** 클릭
2. 기존 자회사 종합관리 엑셀 파일(`.xlsx`) 선택
3. **가져오기 실행** 클릭

아래 시트를 자동으로 인식해서 분류·저장합니다.

| 엑셀 시트 | 저장되는 곳 |
|-----------|-------------|
| 매출(과세), 매출(면세) | 매출 세금계산서 |
| 과세계(입), 면세계(입) | 매입 세금계산서 |
| 우리카드및기타지출 | 카드 사용내역 |
| 입금통장, 출금통장 | 통장 입출금 |

> 💡 명령줄에서 한 번에 가져오려면:
> ```bash
> npm run import:excel -- "엑셀파일경로.xlsx"
> ```

## 🛠️ 기술 구성 (참고용)

- **Next.js 14** (프론트엔드 + 백엔드 통합)
- **TypeScript / Tailwind CSS**
- **Prisma + SQLite** (파일 기반 데이터베이스 — `prisma/erp.db`)
- **SheetJS(xlsx)** 엑셀 파싱

### 자주 쓰는 명령어

| 명령 | 설명 |
|------|------|
| `npm run dev` | 개발 모드 실행 (평소 사용) |
| `npm run build` 그리고 `npm start` | 운영 모드 실행 (더 빠름) |
| `npm run db:studio` | 데이터베이스를 표로 직접 보기/편집 |
| `npm run import:excel -- "파일.xlsx"` | 엑셀 일괄 가져오기 |

## 💾 백업 / 이전

- **백업**: `prisma/erp.db` 파일을 복사해서 보관하세요.
