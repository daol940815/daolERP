# daolERP — (주)스피어스 자회사 종합관리 웹

기존 엑셀로 관리하던 **자회사 종합관리 문서**를 웹 시스템으로 옮긴 프로젝트입니다.
세금계산서 · 카드 사용내역 · 통장 입출금 · 영업이력 등 **로우데이터 입력/조회**, **엑셀 가져오기**,
**자금현황/영업 대시보드**, **로그인 보안**을 제공합니다.

> 데이터는 **PostgreSQL(클라우드 DB)** 에 저장됩니다. 인터넷 배포로 동료와 공유할 수 있습니다.

## 🖥️ 로컬에서 실행하기 (개발/테스트용)

### 1단계. Node.js 설치 (최초 1회)

1. 인터넷에서 **[https://nodejs.org](https://nodejs.org)** 접속
2. 가운데 **LTS** 버튼(권장 버전)을 눌러 설치 파일 다운로드
3. 다운로드한 파일을 실행해서 **"다음 → 다음 → 설치"** 클릭
4. 확인: 터미널에서 `node -v` → `v20...` 이상이면 성공

### 2단계. PostgreSQL 준비

운영과 동일하게 PostgreSQL 을 사용합니다. 둘 중 하나:
- **간편**: [Neon](https://neon.tech) 무료 DB 주소를 그대로 사용 (별도 설치 불필요)
- **로컬 설치**: PC에 PostgreSQL 설치 후 데이터베이스 1개 생성

### 3단계. 환경변수 파일 만들기

`.env.example` 을 복사해 **`.env`** 파일을 만들고 값을 채웁니다.
(`DATABASE_URL`, `DIRECT_URL`, `AUTH_SECRET`, `ADMIN_USERNAME`, `ADMIN_PASSWORD`)

### 4단계. 준비 & 실행

```bash
npm install          # 필요한 프로그램 설치 (1~2분)
npm run db:setup     # 표(테이블) 생성 + 관리자 계정 생성
npm run dev          # http://localhost:3000
```

브라우저에서 `http://localhost:3000` 접속 → **로그인**(.env 의 아이디/비밀번호) → 사용 시작.

> 종료: 터미널에서 `Ctrl + C`. 다음부터는 `npm run dev` 만 실행하면 됩니다.

## 📥 기존 엑셀 데이터 가져오기

1. 왼쪽 메뉴에서 **⬆️ 엑셀 가져오기** 클릭
2. 기존 자회사 종합관리 엑셀 파일(`.xlsx`) 선택
3. **가져오기 실행** 클릭

아래 시트를 자동으로 인식해서 분류·저장합니다.

| 엑셀 시트 | 저장되는 곳 |
|-----------|-------------|
| 매출(과세), 매출(면세) | 매출 세금계산서 |
| 과세계(입), 면세계(입) | 매입 세금계산서 |
| 매출이력 | 영업/매출이력 |
| 우리카드및기타지출 | 카드 사용내역 |
| 입금통장, 출금통장 | 통장 입출금 |

> 💡 명령줄에서 한 번에 가져오려면:
> ```bash
> npm run import:excel -- "엑셀파일경로.xlsx"
> ```

## 🛠️ 기술 구성 (참고용)

- **Next.js 14** (App Router, 프론트엔드 + 백엔드 통합)
- **TypeScript / Tailwind CSS**
- **Prisma + PostgreSQL** (운영: Neon / 로컬: PostgreSQL)
- **인증**: jose(세션 JWT) + bcryptjs(비밀번호 해시), 미들웨어로 전 경로 보호
- **SheetJS(xlsx)** 엑셀 파싱 (서버에서 메모리 처리 — 디스크 저장 없음)

### 자주 쓰는 명령어

| 명령 | 설명 |
|------|------|
| `npm run dev` | 개발 모드 실행 (평소 사용) |
| `npm run db:setup` | 마이그레이션 적용 + 관리자 계정 생성 |
| `npm run db:migrate` | 스키마 변경 시 새 마이그레이션 생성 |
| `npm run db:seed` | 관리자 계정만 생성/갱신 |
| `npm run db:studio` | 데이터베이스를 표로 직접 보기/편집 |
| `npm run import:excel -- "파일.xlsx"` | 엑셀 일괄 가져오기 |

### 환경변수 (`.env` — 깃에 올라가지 않음)

`DATABASE_URL`, `DIRECT_URL`, `AUTH_SECRET`, `ADMIN_USERNAME`, `ADMIN_PASSWORD`, `ADMIN_NAME`
→ 자세한 설명은 `.env.example` 참고.
